# greatscott-ai-crawl-files

Great Scott AI Crawl Files

This repository hosts machine-readable structured content for the Great Scott website.

Files included:
- llms.txt → summary layer for AI systems
- llms-full.txt → expanded semantic business context
- robots.txt → crawler instructions and sitemap reference

Purpose:
Improve discoverability and structured understanding of the Great Scott website for search engines and AI systems.

Live endpoint:
https://files.callusgreatscott.com/

Structured endpoints:
- /llms.txt
- /llms-full.txt
- /robots.txt
